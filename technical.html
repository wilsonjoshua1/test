Youtube to Spotify

I created this program to convert a youtube playlist into a Spotify playlist. I did this because when I was younger I did not own a spotify account and I created personal playlist for myself. As I got older I started to use spotify more and I did not want to recreate these playlist again on Spotify. Especially since these playlists are a couple of hours long. I created a python script that will use the Spotify API and Youtube API to convert a Youtube Playlist to a Spotify one.
